# Business Planner — Архитектура Фронтенда

## Обзор

Фронтенд построен на React 19 с TypeScript, использует Vite для сборки и TailwindCSS для стилизации. Приложение следует современным практикам разработки с использованием хуков, контекстов и компонентного подхода.

## Технологический стек

| Слой | Технология | Зачем используется |
|------|------------|-------------------|
| Framework | React 19.2.4 | Библиотека для построения пользовательского интерфейса |
| Language | TypeScript 6.0.2 | Типизация для надёжности кода |
| Build Tool | Vite 8.0.4 | Быстрая сборка и HMR |
| Styling | TailwindCSS 4.2.2 | Utility-first CSS фреймворк |
| Routing | React Router DOM 7.14.1 | Клиентская маршрутизация |
| HTTP Client | Axios 1.15.0 | Выполнение HTTP-запросов к API |
| Rich Text Editor | TipTap 3.23.4 | Редактор форматированного текста |
| Drag & Drop | @dnd-kit 6.3.1 | Библиотека для drag-and-drop |
| Charts | Recharts 3.8.1 | Библиотека для графиков |
| Icons | Lucide React 1.8.0 | Иконки |
| Notifications | react-hot-toast 2.6.0 | Toast-уведомления |
| Markdown | react-markdown 10.1.0 | Рендеринг Markdown |

## Основные компоненты

### 1. API Клиент (api.ts)

Централизованный клиент для выполнения HTTP-запросов к бэкенду.

**Основные функции:**
- Настройка axios с базовым URL и интерцепторами
- Автоматическая добавка Bearer токена в заголовки
- Обработка ошибок и редирект на страницу авторизации при 401
- Типизированные функции для всех API-эндпоинтов

**Типы данных:**
- `User`, `BusinessPlan`, `PlanBlock`, `FinancialChart`, `ChartPoint`, `Currency`
- Типы для создания/обновления сущностей

### 2. Страницы (pages/)

#### AuthPage.tsx
Страница авторизации с формами входа и регистрации.

**Функционал:**
- Переключение между входом и регистрацией
- Валидация форм
- Сохранение токена в localStorage
- Редирект на дашборд после успешной авторизации

#### BusinessPlansPage.tsx
Страница со списком бизнес-планов пользователя.

**Функционал:**
- Отображение карточек бизнес-планов
- Создание нового плана через модальное окно
- Переход к деталям плана
- Удаление плана

#### BusinessPlanDetailsPage.tsx
Страница с деталями бизнес-плана и его блоками.

**Функционал:**
- Редактирование названия и описания плана
- Drag-and-drop для перестановки блоков
- Создание, редактирование, удаление блоков
- Система черновиков для блоков
- Связь блоков с финансовыми графиками
- Экспорт в Excel/CSV

#### FinancialPlansPage.tsx
Страница со списком финансовых планов пользователя.

**Функционал:**
- Отображение карточок финансовых планов
- Создание нового плана с выбором валюты
- Переход к деталям плана
- Удаление плана

#### FinancialPlanDetailsPage.tsx
Страница с деталями финансового плана и графиком.

**Функционал:**
- Редактирование названия, описания и валюты плана
- Отображение графика с таймфреймами (Неделя, Месяц, 3 месяца, Год)
- Агрегация данных по выбранному таймфрейму
- Создание, редактирование, удаление точек графика
- Экспорт в Excel/CSV

#### ProfilePage.tsx
Страница профиля пользователя.

**Функционал:**
- Отображение информации о пользователе
- Редактирование профиля
- Изменение пароля

### 3. Компоненты (components/)

#### RichTextEditor.tsx
Редактор форматированного текста на базе TipTap.

**Функционал:**
- Форматирование текста (жирный, курсив, подчёркивание)
- Списки (маркированные, нумерованные)
- Заголовки
- Ссылки
- Изображения
- Таблицы
- Подсветка кода
- Зачёркивание
- Подстрочный и надстрочный индексы

#### BlockRenderer.tsx
Компонент для рендеринга блоков бизнес-плана.

**Функционал:**
- Отображение разных типов блоков (текст, заголовок, список, и т.д.)
- Рендеринг Markdown
- Отображение прикреплённых медиа

#### Sidebar.tsx
Боковая панель навигации.

**Функционал:**
- Навигация между разделами приложения
- Отображение активного раздела
- Кнопка выхода

#### Modal компоненты
- `AuthModal.tsx` — Модальное окно авторизации
- `BlockModal.tsx` — Модальное окно для создания/редактирования блоков
- `ConfirmModal.tsx` — Модальное окно подтверждения действий
- `LogoutConfirmModal.tsx` — Модальное окно подтверждения выхода
- `PointModal.tsx` — Модальное окно для создания/редактирования точек графика
- `SettingsModal.tsx` — Модальное окно настроек

### 4. Фичи (features/)

#### auth/
Контекст и хуки для аутентификации.

**Функционал:**
- Хранение токена и данных пользователя
- Функции для входа, выхода, регистрации
- Проверка авторизации

#### theme/
Контекст для управления темой (тёмная/светлая).

**Функционал:**
- Переключение темы
- Сохранение темы в localStorage
- Применение темы к приложению

### 5. Общие ресурсы (shared/)

#### theme.ts
Объект с общими стилями и темой.

**Содержит:**
- Цветовые схемы для тёмной и светлой темы
- Стили кнопок, инпутов, карточек
- Утилиты для TailwindCSS

## Основные функции для пользователей

### Бизнес-планы

#### Создание бизнес-плана
1. Нажмите кнопку "Создать" на странице бизнес-планов
2. Введите название и описание
3. Нажмите "Сохранить"

#### Редактирование бизнес-плана
1. Откройте детали бизнес-плана
2. Нажмите кнопку "Редактировать"
3. Измените название или описание
4. Нажмите "Сохранить"

#### Добавление блоков
1. Откройте детали бизнес-плана
2. Нажмите кнопку "Добавить блок"
3. Выберите тип блока
4. Введите содержимое
5. Нажмите "Сохранить"

#### Перестановка блоков
1. Откройте детали бизнес-плана
2. Перетащите блок на нужное место (drag-and-drop)

#### Связь с финансовыми графиками
1. При редактировании блока выберите финансовые графики для связи
2. Графики будут отображаться в блоке

#### Экспорт бизнес-плана
1. Откройте детали бизнес-плана
2. Нажмите кнопку "Excel" или "CSV"
3. Файл автоматически скачается

### Финансовые планы

#### Создание финансового плана
1. Нажмите кнопку "Создать" на странице финансовых планов
2. Введите название и описание
3. Выберите валюту
4. Нажмите "Сохранить"

#### Добавление точек графика
1. Откройте детали финансового плана
2. Нажмите кнопку "Добавить точку"
3. Выберите тип (Доход/Расход)
4. Введите дату, сумму и описание
5. Нажмите "Сохранить"

#### Просмотр графика
1. Откройте детали финансового плана
2. Выберите таймфрейм (Неделя, Месяц, 3 месяца, Год)
3. График автоматически обновится

#### Редактирование точки
1. Нажмите кнопку "Редактировать" на точке
2. Измените данные
3. Нажмите "Сохранить"

#### Экспорт финансового плана
1. Откройте детали финансового плана
2. Нажмите кнопку "Excel" или "CSV"
3. Файл автоматически скачается

### Профиль

#### Редактирование профиля
1. Откройте страницу профиля
2. Измените имя, фамилию или email
3. Нажмите "Сохранить"

#### Изменение пароля
1. Откройте страницу профиля
2. Введите текущий пароль
3. Введите новый пароль
4. Нажмите "Сохранить"

## Запуск приложения

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview
```

## Переменные окружения

Создайте файл `.env` в корне проекта:

```bash
VITE_API_URL=http://localhost:8000
```

## Сборка

```bash
npm run build
```

Собранные файлы будут в директории `dist/`.
