# Business Planner — Руководство для пользователей

## Что такое Business Planner?

Business Planner — это веб-приложение для планирования бизнеса и управления финансами. Оно помогает создавать бизнес-планы, отслеживать доходы и расходы, строить финансовые графики и экспортировать данные.

## Основные функции

### 1. Бизнес-планы

**Что это такое?**
Бизнес-план — это документ, описывающий вашу бизнес-идею, цели и стратегии. В Business Planner вы можете создавать структурированные планы с блоками разного типа.

**Как это работает?**
- Вы создаёте бизнес-план с названием и описанием
- Добавляете блоки с текстом, заголовками, списками, таблицами
- Можете перетаскивать блоки для изменения порядка
- Связываете блоки с финансовыми графиками для наглядности
- Экспортируете план в Excel или CSV

**Зачем это нужно?**
- Структурировать идеи и планы
- Делиться планами с командой или инвесторами
- Связывать текстовые планы с финансовыми показателями
- Иметь всё в одном месте

### 2. Финансовые планы

**Что это такое?**
Финансовый план — это график, показывающий ваши доходы и расходы во времени. Вы можете добавлять точки (транзакции) и видеть динамику финансов.

**Как это работает?**
- Вы создаёте финансовый план с выбором валюты (RUB, USD, EUR и др.)
- Добавляете точки с датой, типом (доход/расход) и суммой
- Выбираете таймфрейм для просмотра (неделя, месяц, 3 месяца, год)
- Система автоматически агрегирует данные и строит график
- Экспортируете данные в Excel или CSV

**Зачем это нужно?**
- Отслеживать доходы и расходы
- Видеть финансовую динамику
- Планировать бюджет
- Принимать обоснованные решения

### 3. Валюты

**Что это такое?**
Система поддерживает множество валют: фиатные (RUB, USD, EUR, GBP, JPY и др.) и криптовалюты (BTC, ETH и др.).

**Как это работает?**
- Вы выбираете валюту при создании финансового плана
- Курсы валют автоматически синхронизируются с внешними API
- Вы можете просматривать актуальные курсы

**Зачем это нужно?**
- Работать с разными валютами
- Отслеживать курсы в реальном времени
- Планировать в международном масштабе

### 4. Профиль пользователя

**Что это такое?**
Личный кабинет для управления настройками аккаунта.

**Как это работает?**
- Редактирование имени, фамилии, email
- Изменение пароля
- Просмотр информации о профиле

**Зачем это нужно?**
- Поддерживать актуальность данных
- Обеспечивать безопасность аккаунта

## Как пользоваться системой

### Регистрация и вход

1. Перейдите на страницу авторизации
2. Нажмите "Регистрация"
3. Введите email, username и пароль
4. Подтвердите email по ссылке в письме
5. Войдите в систему

### Создание бизнес-плана

1. Нажмите "Создать" на странице бизнес-планов
2. Введите название и описание
3. Нажмите "Сохранить"
4. Добавьте блоки с содержимым
5. Сохраните изменения

### Создание финансового плана

1. Нажмите "Создать" на странице финансовых планов
2. Введите название и описание
3. Выберите валюту
4. Нажмите "Сохранить"
5. Добавьте точки с доходами и расходами
6. Выберите таймфрейм для просмотра графика

### Экспорт данных

1. Откройте детали плана
2. Нажмите кнопку "Excel" или "CSV"
3. Файл автоматически скачается

## Преимущества системы

### Для бизнеса
- **Структурирование:** Все планы и финансы в одном месте
- **Визуализация:** Графики и диаграммы для наглядности
- **Гибкость:** Разные типы блоков и таймфреймы
- **Экспорт:** Возможность выгрузки данных в Excel/CSV

### Для пользователей
- **Простота:** Интуитивный интерфейс
- **Мультивалютность:** Работа с разными валютами
- **Безопасность:** Защищённые данные и авторизация
- **Доступность:** Работает в любом браузере

## Технические детали (для интересующихся)

### Архитектура
- **Бэкенд:** FastAPI (Python) — быстрый и надёжный сервер
- **Фронтенд:** React + TypeScript — современный интерфейс
- **База данных:** PostgreSQL — хранение данных
- **API:** RESTful — стандартный протокол обмена данными

### Безопасность
- JWT токены для авторизации
- Хеширование паролей
- Защищённые HTTP-запросы
- Валидация данных на сервере

### Интеграции
- ExchangeRate-API — курсы фиатных валют
- CoinGecko API — курсы криптовалют
- Email-провайдеры — уведомления

## Часто задаваемые вопросы

**В: Мои данные в безопасности?**
О: Да, все данные защищены, пароли хешируются, используется HTTPS.

**В: Можно ли работать офлайн?**
О: Нет, приложение требует интернет-соединение для работы с сервером.

**В: Поддерживаются ли мобильные устройства?**
О: Да, интерфейс адаптивен и работает на мобильных устройствах.

**В: Как часто обновляются курсы валют?**
О: Курсы обновляются вручную через кнопку синхронизации или по расписанию.

**В: Можно ли удалить свой аккаунт?**
О: Да, вы можете удалить аккаунт в настройках профиля.
